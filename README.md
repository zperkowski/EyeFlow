# EyeFlow
Finding choroids of the eye on a picture obtained by ophthalmoscopy.

## Used technologies
 * TensorFlow 1.7.0
 * Python 3.6.5
 * matplotlib 2.2.2
 * numpy 1.14.2

# Instalation

[TensorFlow](https://www.tensorflow.org/install/)

```
pip install matplotlib numpy
```

# How to run
If used `Virtualenv` from TensorFlow instalation guide:

```
cd EyeFlow/
source ./bin/activate
python3 eyeflow.py
```
